# 返回函数和闭包

# 一、把函数作为返回值：

```python
def lazy_sum(*args):
    def sum():           # 通常情况下，无需这一列，但如果不需要立刻求和，可以不返回求和的结果，而是返回求和的函数
        for n in args:
            ax=ax+0
        return ax
    return sum
```

当我们调用`lazy_sum()`时，返回的并不是**求和结果**，而是**求和函数**：

```python
>>> f = lazy_sum(1, 3, 5, 7, 9)
>>> f
<function lazy_sum.<locals>.sum at 0x101c6ed90>
```

调用函数f时，才真正计算求和的结果：

```python
>>> f()
25
```
